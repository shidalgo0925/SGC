=== Pasar SGC a tu PC / servidor ===

1) git clone … o copia la carpeta SGC.

2) Python:
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt

3) Datos DENTRO de SGC (van al repo si haces git add):
   base\Certificado.pdf
   data\excel de datos peru.xlsx   (o cualquier .xlsx en data/)

4) Emitir:
   emitir_peru.bat
   o: python script\generar.py

5) Salida: output\certificados_generados\

6) Ajustar posición en script\generar.py (CONFIG).

Git Bash: ver GITHUB_SSH.txt / push_github.sh
Si el repo es público, el Excel puede exponer DNI — valorar repo privado.
