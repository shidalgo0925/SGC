=== Pasar SGC a tu PC (Windows) ===

1) Copia la carpeta SGC completa (o el archivo SGC_portable.tar.gz y descomprime).

2) Abre CMD o PowerShell DENTRO de la carpeta SGC (Git Bash: ver nota abajo):
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt

3) Archivos de emisión (junto a la carpeta SGC):
   ..\SGC_Base\Certificado.pdf
   ..\SGC_Base\excel de datos peru.xlsx
   (QR obligatorio en cada PDF; sin subir nada. Sin -u el QR es texto local id/cip/nombre.)

4) Ejecuta:
   emitir_peru.bat
   o: python script\generar.py

5) Los PDFs salen en: output\certificados_generados\

6) Ajusta posición nombre/CIP en script\generar.py (CONFIG).

Opciones (ejemplos):
  python script\generar.py -b base\mi_plantilla.pdf -e data\lista.xlsx -o output\mi_lote
  python script\generar.py --qr-base-url https://ejemplo.com/verificacion
  python script\generar.py -n 1

Checklist: venv + deps -> PDF + Excel -> calibrar CONFIG -> lote 100+ filas -> documentar coordenadas finales.
Sin Python en Git Bash:   bash instalar_python_gitbash.sh
  (usa winget, o descarga el instalador oficial a Descargas y lo ejecuta en silencio)

Git Bash en Windows: el comando "python" a menudo apunta a la Tienda (falla).
  - Crear venv con: py -3 -m venv .venv
  - Luego: ./emitir_cert1.sh   (usa .venv\Scripts\python.exe)
  - O ejecutar emitir_cert1.bat desde CMD.
